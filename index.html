User Agreement & Privacy Policy

Please carefully read and fully understand the User Agreement and Privacy Policy.

To provide core features such as Bluetooth device discovery, connection, parameter reading/writing, and network communication, the app requests only necessary permissions:

1) Bluetooth permissions (for BLE scan/connect);

2) Location permission (required only on some Android versions for BLE scanning);

3) Network permission (for API communication).

You can manage permissions at any time in system settings. We do not request camera permission in this release.
